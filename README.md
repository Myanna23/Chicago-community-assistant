# Chicago Community Environmental Assistant

A Claude-powered chatbot helping Chicago residents navigate 
environmental complaints, health resources, and community 
organizations, built to support communities disproportionately 
impacted by industrial pollution.

## What It Does
- Answers questions about filing environmental complaints in Chicago
- Explains health effects linked to industrial pollution exposure
- Identifies neighborhoods with high environmental complaint density
- Connects residents to workforce, health, and advocacy resources

## How It Works
Built with HTML and JavaScript using the Anthropic Claude API 
(claude-sonnet-4-6). Conversations are context-aware, the assistant 
remembers what was said earlier in the chat.

## How To Use
1. Download or clone this repo
2. Open chicago-chatbot.html in any browser
3. Enter your Anthropic API key
4. Ask anything or tap a topic button to get started
